# Исходные данные

В этой папке находятся все файлы, предоставленные организаторами хакатона. Структура папок сохранена в исходном виде.

## Содержание

- `Доп. материалы/` — учебники и дополнительные документы по флотации.
- `Пример 1/` — данные по КГМК: Гипотезы экспертов и отчет по хвостам.
- `Пример 2/` — данные по НОФ (вкрапленные руды). Гипотезы экспертов и отчет по хвостам.
- `Пример 3/` — данные по НОФ (медные руды). Гипотезы экспертов и отчет по хвостам.
- `Пример 4/` — данные по ТОФ. Гипотезы экспертов и отчет по хвостам.
- `Регламенты/` — технологические регламенты работы оборудования.
- `Схемы флотации/` — технологические схемы обогатительных фабрик.
- `Как читать отчет института по хвостам.docx` — пояснение к структуре Excel-файлов.
